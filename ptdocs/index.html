RESEARCH

Execution environments are the center of computing platforms, such as traditional operating systems, web platforms, and mobile systems. My research has focused on securing execution environments through introducing new security primitives and developing new techniques for debugging and attack diagnosis.

For new security primitives, I developed techniques to transparently isolate untrusted programs in Linux systems [5], and techniques to confine untrusted JavaScript in the browser environment [4]. From such experiences, I rethink the way binary components work in a program, and designed solutions to isolate untrusted program components to prevent backdoors [3] and unexpected side-effects in system and memory [2].

For debugging and attack diagnosis, I analyzed recent advanced memory-error exploits and created techniques to diagnose and classify them through a type system on binaries [7]. To efficiently discover software errors and debug such errors, I developed solutions to systematically detect errors [9] and debug errors [8] using symbolic execution on binaries.

TEACHING

I lead the effort of redesigning the system security curriculum in NUS. I created the undergraduate CS4238 Computer Security Practice, to train the system security skills of senior undergraduate students (Sample Syllabus). To support the instruction of this module, I designed the layout and networking of a new teaching lab, supporting 40 students (10 project groups).

I also redeveloped the undergraduate course CS3235 (Introduction to Computer Security), shifting its focus from coding theory and cryptography to overall security (including basic cryptography, software security, system security, network security, and web security). I also redesigned the graduate course CS5231, from Cryptographic techniques & Data Security to Systems Security.

I have been practicing a new method of teaching security principles through practice. I designed a series of live demos in class, showing students complex concepts like buffer overflow attacks and rootkit analysis. I also distribute virtual machines and external hard drives to students for them to study security tools. The method is effective in motivating students to see security principles in detailed system knowledge.
